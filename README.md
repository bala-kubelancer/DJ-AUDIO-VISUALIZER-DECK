## To run this app

-- Clone 
```
git clone https://github.com/bala-kubelancer/DJ-AUDIO-VISUALIZER-DECK.git
```
-- Install 
```
npm ci
```
-- Run
```
npm run dev
```

## Containerization

```
docker build -t dj-eq-visualizer:latest .
docker run -p 8080:80 dj-eq-visualizer:latest
```
### open http://localhost:8080
